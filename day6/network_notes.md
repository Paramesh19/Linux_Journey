## day6 network commands

- to know ip address
	- ip a
	- ipconfig
- to test internet connectivity
	- ping 10.0.2.15
- check open ports or connections
	- ss -tuln
		- t tcp
		- u udp
		- l listening
		- numbers only
- Download files from terminal
	- wget <url name>
	- eg: wget https://example.com
	- curl <url name>
- for remote login
	- ssh user@ipaddress
