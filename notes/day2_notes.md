## Day2 Commands

- Present working Directory  
	 pwd
- To List all files and folders in a directory
	ls
	ls -a to list hidden files
	ls -la detailed view of all files
- To Change Directory
	cd	
	cd ..   	go back
	cd ~  		 Home Directory
	cd /etc 	 open Linux system configurations
	cd - 		previous folder
- create folders
	mkdir <folder name>	create a new folder
- create files
	touch <file name>	
- To copy/move/delete
	cp <file name> <file name>    copy
	mv <file name> <file name>   move
	rm <file name>		delete file
	rm -r <folder name>	delete folder
- To view file contents
	nano <file name>	To write to the file
	cat <file name>		To display file contents in the terminal
